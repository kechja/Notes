JQuery Mobile Notes

Overview

JQuery Mobile is a mobile web framework. The last major release, version 1.4, was released in 2014, and their blog indicates version 1.5 is in the works, albeit with delays. Top competitors to jQuery mobile at the time of this writing (early 2017) are Bootstrap, PhoneGap, Apache Cordova, and Ionic.

A particular strength of the framework is the ease of setting up page transitions, similar to those of the iphone.

How To Setup

           # Initialize jQuery in the head
              <meta name="viewport" content="width=device-width, initial-scale=1">
              <link rel="stylesheet" href="https://code.jquery.com/mobile/1.4.5/jquery.mobile-1.4.5.min.css">
              <script src="https://code.jquery.com/jquery-1.11.3.min.js"></script>
              <script src="https://code.jquery.com/mobile/1.4.5/jquery.mobile-1.4.5.min.js"></script>
            
           # Put your content in a "page" data-role div
              <div data-role="page" id="mycontent">
                ... your content ...
              </div>

Layout

Divs with data-role="page" wrap all elements, and typically additional divs with data-role="header|main|footer" demarcate those page elements. Note that header and footer create a toolbar style, which may not be what you want.

Additionaly adding class="ui-content adds extra padding and margin.

Transition: Slide Right

Slide Right
data-transition="slide" [data-direction="reverse"] 
             # Add data-transition to any *link*. 
                <a href="#settings" data-transition="slide">Slide Right</a>

             # Reverse using data-direction reverse.
                <a href="#settings" data-transition="slide" data-direction="reverse">Slide Left</a>
